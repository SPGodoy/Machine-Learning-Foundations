# Machine Learning Foundations

## Overview

This repository is designed to build a strong foundation in machine learning by implementing key algorithms and workflows from scratch and with industry-standard libraries.

The focus is not just on using models, but on understanding:
- How they work
- When to use them
- How to evaluate their performance

## Concepts Covered

- Supervised Learning
- Linear Regression
- Logistic Regression
- Decision Trees
- K-Nearest Neighbors (KNN)
- Train/Test Splitting
- Feature Scaling
- Model Evaluation (MSE, MAE, R², Accuracy)
- Overfitting vs Generalization
